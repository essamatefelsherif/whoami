SpotLight ScreenSaver

Tech Stack  C, Win32 API's
URL github  githubLink

A basic Windows Screen Saver written in C using Win32 API's.

The Screen Saver implements the famous After Dark Screen Saver Spotlight module, where the desktop becomes dark/black and parts are "illuminated" by a randomly moving light spot.

If you only have one monitor you have nothing to worry about, but those with two or more may notice oddities.
