Leaflet themeControls

Tech Stack  JavaScript, HTML, CSS, Leaflet.js
URL website  websiteLink
URL github  githubLink

A set of themeControls that extend the Leaflet.js native controls, and other additional controls, in order to customize the on-screen shape of the control as defined by the associated css and image files provided.

Using themeControls plugin within a web map will enhance the user interface with a theme of cultural related icons.

This plugin was tested against Leaflet versions greater than 1.0, up to 1.7.1.
